> _NB: If you're switching to [lazy](https://github.com/folke/lazy.nvim) like me
> from [packer](https://github.com/wbthomason/packer.nvim), you'd have to delete your
> `~/.config/nvim/plugins/packer_compiled.lua` file, and also your `~/.local/share/nvim/site/pack/packer`
> directory in order for lazy to work correctly_.

> So, run:
> ```sh
> rm -rf ~/.config/nvim/plugins/packer_compiled.lua ;
> rm -rf ~/.local/share/nvim/site/pack/packer ;
> ```
